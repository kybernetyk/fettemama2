Du verspuerst also die Lust deine Meinung einer breiten Oeffentlichkeit kund zu tun? Du willst einen Blogkommentaar hinterlassen? Vieles einfacher als das!

Damit Lamer, Internetausdrucker und Spambots keine Chance haben, kann man Kommentare auf fefemama nur ueber das Telnet interface hinterlassen. Also telnete an fettemama.org port 1337 und gib help ein. Der Rest erklaert sich von selber.

% telnet fettemama.org 1337
Connected to fettemama.org.
Escape character is '^]'.
fettemama.org blog system version v0.2

There are 12 users active.
